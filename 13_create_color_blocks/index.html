Create color blocks :

Etape 1
créer une mini application contenant :
Un champs input number,
un champs input color et
un bouton de validation.
Au click sur le bouton, on créé autant de div(s) qu'indiqué dans le champs number (1 par défaut).
Le(s) block(s) créé(s) aura(ont) la couleur indiquée dans le champs color

Etape 2
au click sur chaque block : passer de l'état active/inactif
les blocks actifs peuvent être mis à jour (on change la couleur)...
au click sur le bouton de validation.

Idée d'amélioration : créer un bouton reset qui vide le contenu de blocks. Si il y a des blocks actifs, ne supprimer que ceux là.

Indices :

  • document.createElement
  • element.appendChild
  • element.value
  • element.style
  • element.onclick
